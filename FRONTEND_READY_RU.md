## 🔌 Подключенные API Endpoints

### Авторизация
```
POST   /register     - Создать аккаунт
POST   /login        - Логин (получить JWT токен)
GET    /me           - Информация текущего пользователя
```

### Игроки (7 endpoints)
```
GET    /api/player/{id}/details      - Информация игрока
GET    /api/player/{id}/stats        - Статистика
GET    /api/player/{id}/fixtures     - История матчей
GET    /api/player/{id}/teams        - История команд
GET    /api/player/favorite          - Избранные (защищено)
POST   /api/player/{id}/favorite     - Добавить в избранное (защищено)
DELETE /api/player/{id}/favorite     - Удалить из избранного (защищено)
```

### Команды (8 endpoints)
```
GET    /api/team/{id}/details        - Информация команды
GET    /api/team/{id}/stats          - Статистика команды
GET    /api/team/{id}/next_game      - Следующий матч
GET    /api/team/{id}/standings      - Место в турнире
GET    /api/team/{id}/players        - Состав команды
GET    /api/team/{id}/fixtures       - История матчей
GET    /api/team/{id}/manager        - Менеджер команды
GET    /api/team/{id}/players_stats  - Игроки со статистикой
+ Endpoints для избранного
```

### Турниры (7 endpoints)
```
GET    /api/tournament/{id}/details        - Информация турнира
GET    /api/tournament/{id}/table          - Таблица
GET    /api/tournament/{id}/stats/teams    - Статистика команд
GET    /api/tournament/{id}/stats/players  - Статистика игроков
GET    /api/tournament/{id}/fixtures       - Все матчи
+ Endpoints для избранного
```

### Менеджеры (5 endpoints)
```
GET    /api/manager/{id}/details    - Информация менеджера
GET    /api/manager/{id}/stats      - Статистика
GET    /api/manager/{id}/teams      - Управляемые команды
GET    /api/manager/{id}/fixtures   - История матчей
+ Endpoints для избранного
```

### Матчи (4 endpoints)
```
GET    /api/fixture/{id}/details           - Информация о матче
GET    /api/fixture/{id}/stats/players     - Статистика игроков
GET    /api/fixture/{id}/stats/goalies     - Статистика вратарей
GET    /api/fixture/{id}/stats/teams       - Статистика команд
```

### Остальное
```
GET    /api/search?q=query    - Глобальный поиск
GET    /api/analytics/*       - Аналитика
```
